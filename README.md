Agrinic: An Organic Farming
📖 Overview

Agrinic is a mobile application designed to support farmers in adopting organic farming practices without the use of chemical fertilizers. The app provides step-by-step crop guides, seasonal crop suggestions, real-time weather updates, and market price information to help farmers make informed decisions. By combining technology and sustainable agriculture, Agrinic empowers farmers to increase yield, reduce waste, and promote eco-friendly farming practices.

🎯 Objectives

Provide a user-friendly mobile platform for organic farming guidance.

Enable farmers to access crop-specific step-by-step guides.

Offer seasonal crop suggestions to optimize productivity.

Display real-time weather information for better farming decisions.

Share market price updates to help farmers sell their produce at fair rates.

✨ Features

📋 Organic Crop Guide – Step-by-step cultivation instructions for various crops.

🌾 Seasonal Crop Suggestions – Recommended crops for each month.

🌦️ Weather Information – Real-time weather updates using OpenWeatherMap API.

💰 Market Prices – Current crop prices to aid in selling produce.

👤 User Profile – Personalized dashboard for farmers.

🔒 Secure Authentication – Email/mobile registration and OTP verification.

🛠️ Tech Stack

Frontend: Flutter (Dart)

Backend: Firebase (Authentication & Firestore)

APIs: OpenWeatherMap API for weather data

Platform: Android/iOS mobile app

⚙️ Installation & Setup

Clone this repository:

git clone https://github.com/your-username/agrinic-app.git


Open the project in Flutter-supported IDE (VS Code or Android Studio).

Install dependencies:

flutter pub get


Connect Firebase project (Authentication & Firestore).

Run the app on an emulator or physical device:

flutter run
